# chatTTS 音色库 TTSlist.com
chatTTS 音色库 /AI声音编号库/音色抽卡不烦恼


## 简介

chatTTS非常好玩，但音色抽卡我花了好多时间，干脆，就全生成了，分享给大家。

下面1000个音色从100开始，步长100，一直生产到 100000。目前还没精力区分男生女生。 选择你喜欢的音色编号，填入你的 chatTTS 参数生产即可。

"temperature": 0.3

"top_p": 0.7
